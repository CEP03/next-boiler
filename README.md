ᮒ᮪ᮠᮤᮞ᮪ ᮄᮞ᮪ ᮃ [Next.js](https://nextjs.org) ᮕᮢᮧᮏᮨᮎ᮪ᮒ᮪ ᮘᮧᮇᮒ᮪ᮞ᮪ᮒᮢᮕ᮪ᮕᮨᮓ᮪ ᮝᮤᮒ᮪ᮂ [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## ᮌᮨᮒ᮪ᮒᮤᮀ ᮞ᮪ᮒᮁᮒᮨᮓ᮪

ᮖᮤᮁᮞ᮪ᮒ᮪, ᮛᮥᮔ᮪ ᮒ᮪ᮠᮨ ᮓᮨᮗᮨᮜᮧᮕ᮪ᮙᮨᮔ᮪ᮒ᮪ ᮞᮨᮁᮗᮨᮁ:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

ᮇᮕᮨᮔ᮪ [http://localhost:3000](http://localhost:3000) ᮝᮤᮒ᮪ᮂ ᮚᮧᮅᮁ ᮘᮢᮧᮝ᮪ᮞᮨᮁ ᮒᮧ ᮞᮨᮈ ᮒ᮪ᮠᮨ ᮛᮨᮞᮥᮜ᮪ᮒ᮪.

ᮚᮧᮅ ᮎᮔ᮪ ᮞ᮪ᮒᮁᮒ᮪ ᮈᮓᮤᮒᮤᮀ ᮒ᮪ᮠᮨ ᮕᮌᮨ ᮘ᮪ᮚ᮪ ᮙᮧᮓᮤᮖᮡᮤᮀ `app/page.tsx`. ᮒ᮪ᮠᮨ ᮕᮌᮨ ᮃᮅᮒᮧ-ᮅᮕ᮪ᮓᮒᮨᮞ᮪ ᮃᮞ᮪ ᮚᮧᮅ ᮈᮓᮤᮒ᮪ ᮒ᮪ᮠᮨ ᮖᮤᮜᮨ.

ᮒ᮪ᮠᮤᮞ᮪ ᮕᮢᮧᮏᮨᮎ᮪ᮒ᮪ ᮅᮞᮨᮞ᮪ [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) ᮒᮧ ᮃᮅᮒᮧᮙᮒᮤᮎᮜ᮪ᮜ᮪ᮚ᮪ ᮇᮕ᮪ᮒᮤᮙᮤᮐᮨ ᮃᮔ᮪ᮓ᮪ ᮜᮧᮃᮓ᮪ [Geist](https://vercel.com/font), ᮃ ᮔᮨᮝ᮪ ᮖᮧᮔ᮪ᮒ᮪ ᮖᮙᮤᮜ᮪ᮚ᮪ ᮖᮧᮁ ᮗᮨᮁᮎᮨᮜ᮪.
